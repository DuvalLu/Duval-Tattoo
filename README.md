# Luis C Velasquez

## A webstie for a tattoo studio. Duval Tattoo. Contains tattoo artist portfolios, contact forms to reach specific tattoo artists, hourly rates, and general information about the shop and artists backgrounds.

## When considering a tattoo and visiting the website, you should take some time to click into each respective artists portfolio. Examine their respective work. Once you've found the artist you'd like to get a tattoo from, fill out a form to get in touch.

## Technologies used:

    1. HTML
    2. CSS
    3. BOOTSTRAP
